## 🌟 Overview

**AI Crypto Analyst** is a modern AI-driven cryptocurrency dashboard that combines **real-time market data** with **intelligent AI analysis**.

With the rise of AI-based crypto tools and dashboards, developers are increasingly building systems that turn raw market data into actionable insights. ([Voice of Chain][1])

This project is designed to:

* Simplify crypto analysis
* Provide AI-powered insights
* Deliver a premium user experience

---

## ✨ Features

### 📊 Market Insights

* Real-time crypto prices (CoinGecko API)
* 24h change indicators (📈📉)
* Market status & last updated time

### 🚀 Top Movers

* Top gainers & losers
* Dynamic sorting based on performance

### 🔥 Trending Coins

* Discover trending cryptocurrencies
* Market rank & symbol display

### 🤖 AI Chat Analyst

* ChatGPT-style interface
* Ask anything about crypto
* AI-generated insights

### 🎨 Premium UI/UX

* Glassmorphism design ✨
* Dark futuristic theme 🌌
* Smooth animations
* Chat-style interaction

### ⚡ Smart Backend

* API caching (performance optimized)
* Fallback data system
* Strong error handling

## ⚙️ Installation

### 1️⃣ Clone the repo

```bash
git clone https://github.com/SyedMuqeeth23/AI-Crypto-Analyst.git
cd AI-Crypto-Analyst
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Setup environment

Create `.env` file:

```
OPENAI_API_KEY=your_api_key_here
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```

---

## 🧠 How It Works

* Fetches live crypto data from CoinGecko
* Uses caching to reduce API calls
* Sends user queries to AI agent
* Displays results in a chat interface
